前端 rap 选手

2023/4/14 javascriptvue

输入框文字跳动效果,看着挺有趣的记录一下。

主要有两种思路：

  1. 利用input,div标签,div初始隐藏，input失焦后将div显示，内容赋给div，input隐藏。文字在input里是无法跳动的，所以使用其div配合动画。
  1. 直接使用可编辑的 div。contenteditable 。

我这里使用的是第一种方法,话不多说,开始撸。

先把基本的写好

<style>
    #input {
        outline: none;
        padding: 0 15px;
        border: 1px solid #b4b3b3;
        height: 36px;
        box-sizing: border-box;
        border-radius: 4px;
    }

    /*  文字跳动动画 */
    @keyframes rap {
        25% {
            transform: translate(0, 8px);
        }

        75% {
            transform: translate(0, -8px);
        }
    }
</style>
<body>
    <input placeholder="请输入" id="input" />
</body>

接下来就是样式问题，如何让div与输入框样式保持一致，当然你可以写一个公共样式类来与其绑定，不过这里就不这样做了，这里介绍 一个api

getComputedStyle(element, [pseudoElt])
获取指定节点的最终样式,两个参数
element: 指定节点
pseudoElt: 可选，指定一个要匹配的伪元素的字符串 如 '::after' ,不需要的话通常传一个false, 兼容低版本火狐

我们将其写成一个方法，代码抽离在大多数时刻都是一个好习惯

    const inputEl = document.querySelector('#input');
    inputEl.addEventListener('blur', e => {// 输入框失去焦点
            const { target } = e
            const value = target.value
            if (value === '') return // 值为空不做处理
            let divEl = document.createElement('div')
            cloneElStyle(inputEl,divEl) // 将input的样式复制到div
            divEl.addEventListener('click', e => {// div 点击事件
              
            })
    })

    function cloneElStyle(socurs, target) {
        const inputStyles = getComputedStyle(socurs, false)
        console.log(inputStyles)
        Object.values(inputStyles).forEach(v => {
            target.style[v] = inputStyles.getPropertyValue(v)
        })
    }

打印样式对象

返回的style是一个实时的 CSSStyleDeclaration 对象，当元素的样式更改时，它会自动更新本身。 我们将这些样式复制到div上并替换input,同样的样式在div上发现字体往上偏移了,我们把div行高设置为自身高度 就好了

 const inputEl = document.querySelector('#input');
    inputEl.addEventListener('blur', e => {// 输入框失去焦点
        const { target } = e
        const value = target.value
        if (value === '') return
        let divEl = document.createElement('div')
        cloneElStyle(target, divEl)
        divEl.style.lineHeight = divEl.style.height
        target.parentElement.replaceChild(divEl, target)
        divEl.addEventListener('click', e => {// div 点击事件
            
        })
    })

    function cloneElStyle(socurs, target) {
        const inputStyles = getComputedStyle(socurs, false)
        Object.values(inputStyles).forEach(v => {
            target.style[v] = inputStyles.getPropertyValue(v)
        })
    }

ok,现在只需要把动画给实现就基本完成了
我们把每个字符都用span标签包裹，给span绑定动画实现上下跳动

 function appChildrenSpan(parent, value = '') {
        parent.innerHTML = ''
        for (var i = 0; i < value.length; i++) {
            const spanEl = document.createElement('span')
            spanEl.innerHTML = value[i]
            const delay = -i * 0.1 // 延迟
            spanEl.style.display = 'inline-block'
            spanEl.style.animation = `rap 0.8s linear infinite ${delay}s`
            parent.appendChild(spanEl)
        }
    }

vue 指令

export default {
    install(vue) {
        vue.directive('rap', {
            bind(el, binding) {
                el.addEventListener('blur', EventFn)
            },
            unbind(el, binding) {
                el.removeEventListener('blur', EventFn)
            },

        })
        function EventFn(el) {
            const { target } = el
            const value = target.value
            if (value === '') return
            let divEl = document.createElement('div')
            cloneElStyle(target, divEl)
            divEl.style.lineHeight = divEl.style.height
            appChildrenSpan(divEl, value)
            target.parentElement.replaceChild(divEl, target)
            divEl.addEventListener('click', e => {
                const { target } = e
                el.value = target.innerText
                el.style.display = 'inline-block'
                target.parentElement.replaceChild(el, target)
                el.focus()
                divEl.remove()
                divEl = null
            })
        }
    }
}



function appChildrenSpan(parent, value = '') {
    parent.innerHTML = ''
    for (var i = 0; i < value.length; i++) {
        const spanEl = document.createElement('span')
        spanEl.innerHTML = value[i]
        const delay = -i * 0.1
        spanEl.style.display = 'inline-block'
        spanEl.style.animation = `rap 0.8s linear infinite ${delay}s`
        parent.appendChild(spanEl)
    }
}

function cloneElStyle(socurs, target) {
    const inputStyles = getComputedStyle(socurs, false)
    console.log(inputStyles)
    Object.values(inputStyles).forEach(v => {
        try {
            target.style[v] = inputStyles.getPropertyValue(v)
            // eslint-disable-next-line no-empty
        } catch (e) {
            console.log()
        }
    })
}
